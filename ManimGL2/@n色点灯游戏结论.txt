先把符号分开：设**状态数为 m**，**棋盘边长为 L**。

## 结论

对标准循环点灯规则：

```text
按一次：自身和上下左右都 +1 mod m
目标：全 0 → 全 1
棋盘：L×L 方阵，L 任意
```

则：

```text
对所有 L×L 方阵都必有解 ⇔ m | 4
```

也就是只有：

```text
m = 1, 2, 4
```

必然有解。

| 状态数 m | 是否所有方阵必有解 | 说明                 |
| ----: | --------: | ------------------ |
|     1 |         是 | 平凡情况               |
|     2 |         是 | 普通二色点灯，GF(2)       |
|     4 |         是 | 四色循环 Z₄，方阵全 1 目标成立 |
|  其它 m |         否 | 至少存在某个 L×L 方阵无解    |

这里用的是标准线性模型：把按钮向量写成 `x`，点灯矩阵写成 `A`，目标就是解 `A x = 1 mod m`。这种建模本身就是 Lights Out 的通用线性代数形式。([Puzzling Stack Exchange][1])

## 最小反例：m=3，2×2 已经无解

2×2 棋盘设四个按钮次数为：

```text
a b
c d
```

每个灯都会被 3 个按钮影响，所以方程是：

```text
a+b+c = 1
a+b+d = 1
a+c+d = 1
b+c+d = 1        mod 3
```

两两相减可得：

```text
a=b=c=d
```

代回任意一式：

```text
3a = 1 mod 3
```

即：

```text
0 = 1 mod 3
```

矛盾。

所以 3 状态循环点灯在 2×2 方阵上从全灭到全亮无解。

## 为什么合数也会失败

因为有中国剩余/取模因子关系：

```text
如果 A x = 1 mod m 有解
那么对任意 d | m，也必有 A x = 1 mod d 的解
```

所以只要某个因子已经失败，原来的 m 也失败。

例如：

|  m | 因子 | 失败来自           |
| -: | -: | -------------- |
|  6 |  3 | 2×2 mod 3 无解   |
| 10 |  5 | 11×11 mod 5 无解 |
| 14 |  7 | 3×3 mod 7 无解   |
| 16 |  8 | 9×9 mod 8 无解   |
| 20 |  5 | 11×11 mod 5 无解 |

判断一般模数时，可以先做首行压灯，把 `L²` 个变量压成 `L` 个变量；合数模数下再用 Smith 标准形判断对角同余是否可解，条件是每个对角方程满足 `gcd(m,dᵢ) | sᵢ`。([Puzzling Stack Exchange][1])

## m=7 的小反例：3×3

在 mod 7 下，3×3 有一个静默检验向量：

```text
1 3 1
3 2 3
1 3 1
```

它满足：

```text
A w = 0 mod 7
```

但它所有元素之和为：

```text
1+3+1+3+2+3+1+3+1 = 18 = 4 mod 7
```

如果存在解 `A x = 1`，则：

```text
wᵀ A x = wᵀ 1
```

左边：

```text
wᵀ A x = (A w)ᵀ x = 0
```

右边：

```text
wᵀ 1 = 4
```

矛盾。

所以 7 状态循环点灯在 3×3 方阵上无解。

## 最终一句话

```text
循环 m 状态点灯，标准 L×L 方阵，全 0 → 全 1，
对所有 L 都保证有解，当且仅当 m = 1, 2, 4。
```

其中：

```text
m=2 是 GF(2) 普通点灯；
m=4 是 Z₄ 四色循环；
m=8 已经不行，9×9 就是反例。
```
